# Airport Transfers Booking Website

## Project Features
- User-friendly interface for booking airport transfers.
- Multiple vehicle options for different group sizes.
- Real-time pricing calculator based on selected vehicle and route.
- Secure payment processing with multiple payment gateways.
- Option to schedule transfers in advance.
- User accounts for managing bookings and payment information.

## Technology Stack
- **Frontend:** HTML, CSS, JavaScript, React
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Payment Processing:** Stripe, PayPal

## Installation Steps
1. Clone the repository:
   ```
   git clone https://github.com/JollyTechpro/JollyTech.git
   cd JollyTech
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Set up environment variables (create a `.env` file):
   ```
   DATABASE_URL=your-database-url
   STRIPE_SECRET_KEY=your-stripe-secret-key
   PAYPAL_CLIENT_ID=your-paypal-client-id
   ```
4. Run the application:
   ```
   npm start
   ```

## Pricing Model
- Competitive pricing based on distance and vehicle type.
- Discounts available for round trips and group bookings.
- Transparent pricing with no hidden fees.

## Getting Started Instructions
1. Visit the website on launch.
2. Select your desired pickup and drop-off locations and dates.
3. Choose the vehicle that suits your needs.
4. Proceed to checkout and complete your booking.
5. You will receive a confirmation email with your booking details.
